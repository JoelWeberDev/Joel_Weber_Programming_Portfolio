Development Portfolio

Here are some key stepping stones along my engineering journey.
Sensor Networks Research

(May - Aug 2025)

  • Research with Dr. Ferdouse at Wilfrid Laurier Univeristy exploring how networks of sensors can be used to detect and locate spoiled produce and food in storage.
  • Designed wireless sensor devices and communication protocols for data transmitted between sensors and the central controller.
  • Worked with interfarometers and spectrometers for analyzing gasses to idenifying vapour from spoiled products.
Quantum Algorithm for Legendre Pairs

(May - Aug 2025)

  • Read and experimented with Dr. Ilias Kotsireas's research about the Legendre pairs a combinatorial math structure.
  • Currently working to adapt Grover's quantum search algorithm to accelerate the search for Legendre pairs in an exponentially large space.
  • Writing python code with the IBM qiskit library to simulate and actually run quantum algorithms.
Cellular Automata Simulator

(Jan - Apr 2025)

  • I led a 12 person group project in a software engineering course to research, design, and develop a java desktop application to simulate various cellular automata setups.
  • From what we learned in the course we established and applied robust engineering principles for our design and development.
  • Together with another group member I implemented the entire object oriented java desktop application for cellular automata simulations.
Automated Weeding System

(June 2020 - Present)

  • Since 11th grade I have been working with a local organic farmer to develop a camera to detect crop rows in a field and steer a cultivator to uproot weeds while avoiding the rows.
  • Throughout my ongoing endeavour to create the recognition I have experimented with many computer vision techniques for extracting meaningful results from images in real time.
  • I have been using a raspberry pi along with a realsense depth camera to detect the rows and communicate with the tractor electronics.
Scientific Computing Projects

(Aug 2024 - Apr 2025)

  • Over the past year I have been learning various scientific computing methods and programming languages such as R, Mathematica, Matlab, Python (numpy & scipy), and maple.
  • In this GitHub repository are some examples of scientific computing school assignment along with some curiosity projects.
NAS File Server

(Sept - Dec 2024)

  • When I switched my primary operating system from Windows to Linux I quickly discovered that Windows OneDrive does not mesh well with Linux. So I decided to make my own personal file syncing server.
  • I setup a raspberry pi computer with some external SSD drives to run as my own network attached storage (NAS).
  • I setup a NordVpn mesh network to give myself fully remote access to the file server from any network.
IoT Image Collection System

(July - Aug 2024)

  • During my summer internship at Greentronics Ltd. I created a fully remote and unsupervised camera system to collect depth image data in agricultural applications.
  • I created robust hardware and software controls to handle errors, power losses, startups / shutdowns, and fully remote access with status updates.
  • I collected over 2 TB of image data to be used in product development and testing.
Agricultural Embedded Systems

(May - Aug 2024)

  • I created a real time algorithm to rescale compressed images on an embedded C ISOBUS display.
  • I implemented error checking and handling for display objects in accordance with the ISOBUS standard.
  • Working with limited embedded devices I gained an understanding of various layers of memory and how to develop at both the hardware and software level.
  • I also develop a QT C++ based GUI application for embedded Linux devices.
Building a 3D Printer

(June 2023- Aug 2024)

  • Using salvaged mechanical components and electronics from a photocopier and my custom sheet metal pieces I created the bed and chassis system for a 3 axis 3D printer.
  • I built the printer control system by conscripted an Arduino mega board along with the Ramps 1.4 using the open source Marlin 3D printing software.
  • I create various 3D designs using Fusion 360 and OpenSCAD and used the Universal G-Code Sender (USG) software to slice the print.

Two Player Python Game

(Mar - May 2022)

  • As a high school capstone project I along with 2 other friends built a fully functional python game using the CMU-graphics library.
  • In the development I came up with the initial idea of a 2D ship duelling game, wrote the backend game mechanics, and structured the object oriented design.

Tire Air Pressure Calculator

(Feb 2022)

  • Throughout high school I worked as a technician installing agricultural air and hydraulic systems. So for for another high school project I built a python air flow calculator to determine the amount time required to inflate a tire given various parameters.
  • I used iterative evaluations of the ideal gas law to create the airflow with respect to time curve.
  • Both the front end and back end were implemented using the python CMU-graphics library.
Building RC Airplanes

(Dec - June 2022)

  • Using salvaged electrical components I built a hot wire foam cutter to make airplane wings and fuselages.
  • I wired and adapted the control electronics such as ECM motor drives, transmitters / receivers, cameras, and servo motors.
