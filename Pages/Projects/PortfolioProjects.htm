Crop Row Recognition System

Since early 2020 I have been researching how computer vision has and can be integrated into agricultural systems to increase precision and opertion efficiency. I am currently working to develop a system that can detect crop rows from video and guide a weeding implement accordingly. This project is still in prelimniary stages, but I currently have a model functional enough to most green crops in low weed foliage. My next steps for this project include calculating object size from the camera field of view and employing crops spacing inference to differentiate between crop rows and weeds which are located more sporatically.
